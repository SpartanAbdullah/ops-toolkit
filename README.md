# Ops Toolkit

Modern micro-SaaS operations toolkit for small businesses, warehouses, HR teams, and operations managers.

## Tools
- UAE Overtime Calculator
- SKU Barcode Batch Generator
- Petty Cash Tracker (planned)
- Equipment Maintenance Reminder (planned)

## Tech Stack
Next.js  
TypeScript  
TailwindCSS  
shadcn/ui  

## Vision
Ops Toolkit aims to replace scattered spreadsheets and manual operational workflows with simple focused utilities.
